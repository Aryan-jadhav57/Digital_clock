
# 🕒 Digital Clock (Tkinter)

A simple **Python Tkinter GUI application** that displays the current time and date in real-time.  
The clock updates every second using Python’s `strftime` function and features a colorful, user-friendly interface.

---

## 🚀 Features
- Real-time digital clock with **HH:MM:SS AM/PM format**.
- Displays **current date** alongside time.
- **Auto-refresh every second** using Tkinter’s `after()` method.
- Customizable **font, background, and text colors**.

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Tkinter (GUI), time (strftime)
