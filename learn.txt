## 🕒 Digital Clock Project Explanation

This project creates a simple **digital clock** using Python’s **Tkinter** library for the graphical interface.

### Code Breakdown

1. **Importing Libraries**
   ```python

   import tkinter as tk
   from time import strftime
   ```
   - `tkinter`: Used to build the GUI (Graphical User Interface).
   - `strftime`: A function from the `time` module that formats the current time.

2. **Creating the Main Window**
   ```python

   root = tk.Tk()
   root.title("Digital Clock")
   ```
   - `Tk()` initializes the main application window.
   - `title()` sets the window’s title to "Digital Clock".

3. **Defining the Time Function**
   ```python
   
   def time():
       string = strftime("%H:%M:%S %p \n %D")
       label.config(text=string)
       label.after(1000, time)
   ```
   - `strftime("%H:%M:%S %p \n %D")`: Formats the current time as **Hours:Minutes:Seconds AM/PM** and the **Date**.
   - `label.config(text=string)`: Updates the label with the new time.
   - `label.after(1000, time)`: Calls the `time()` function again after 1000 milliseconds (1 second), ensuring the clock updates continuously.

4. **Creating the Label**
   ```python
   label = tk.Label(root, font=('calibri', 50, 'bold'), background='yellow', foreground='black')
   label.pack(anchor='center')
   ```
   - `Label`: Displays the clock text.
   - `font=('calibri', 50, 'bold')`: Sets the font style and size.
   - `background='yellow'`: Sets the background color.
   - `foreground='black'`: Sets the text color.
   - `pack(anchor='center')`: Places the label in the center of the window.

5. **Starting the Clock**
   ```python
   time()
   root.mainloop()
   ```
   - `time()`: Starts the clock function.
   - `mainloop()`: Keeps the window open and responsive until the user closes it.
